## Tips & Tricks

### Best Practices
1. **Download first** if you want to share on multiple platforms
2. **Copy link** is the most versatile option
3. **WhatsApp** includes the most detail
4. **Twitter/X** is best for public sharing
5. **Instagram** is best for visual sharing

### Troubleshooting

**Download not working?**
- Check your browser's download settings
- Allow downloads from this website
- Check your device storage space

**Copy link says "Failed"?**
- Your browser may block clipboard access
- Use the manual copy option from the alert
- Try a different browser

**WhatsApp/Twitter not opening?**
- Check if you're logged in
- Allow pop-ups from this website
- Try opening the app directly first

**Image quality issues?**
- Images are generated at 2x resolution
- Original size is optimized for social media
- If needed, use browser zoom before downloading

## Frequently Asked Questions

**Q: Can I share multiple times?**
A: Yes! Click any button as many times as you want.

**Q: Does sharing cost money?**
A: No, all sharing features are completely free.

**Q: Will my friends need an account?**
A: No, they can view the website without any account.

**Q: Can I edit the message before sharing?**
A: Yes, on WhatsApp and Twitter you can edit before sending.
